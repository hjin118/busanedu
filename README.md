# 부산시 교육지원청 공지사항

부산시 5개 교육지원청의 공지사항을 실시간으로 확인하는 웹페이지입니다.

## 개요

동래, 북부, 남부, 해운대, 서부 교육지원청의 공지사항을 10일 이내 게시물만 필터링하여 보여줍니다.

## 지원 교육지원청

- 동래교육지원청
- 북부교육지원청
- 남부교육지원청
- 해운대교육지원청
- 서부교육지원청

## 사용 방법

```bash
# 클론
git clone https://github.com/hjin118/busanedu.git

# 실행
index.html 파일을 브라우저에서 열면 됩니다.
```
