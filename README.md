# 부산시 교육지원청 공지사항

부산시 5개 교육지원청의 공지사항을 실시간으로 확인하는 웹페이지입니다.

## 개요

동래, 북부, 남부, 해운대, 서부 교육지원청의 공지사항을 10일 이내 게시물만 필터링하여 보여줍니다.

## 지원 교육지원청

- 동래교육지원청
- 북부교육지원청
- 남부교육지원청
- 해운대교육지원청
- 서부교육지원청

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 서버 실행
npm start
```

서버 실행 후 브라우저에서 http://localhost:3000 에 접속하면 됩니다.

## API

- `GET /api/notices` - 모든 교육지원청 공지사항 조회
- `GET /api/offices` - 교육지원청 목록 조회

## 기술 스택

- Node.js
- Express
- Puppeteer (크롤링)
- HTML5 / CSS3 / JavaScript
